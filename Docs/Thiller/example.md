# 추천 결과 예시 (Top-3 등)

## 1. 추천 예시 (10개)

1. 기생충 → 내부자들
- 주요 공통 요소: 사회 부조리·계층 갈등 묘사 / 풍자적 연출 톤 유사 / 현실 비판적 메시지
- 적용 규칙: ②③⑤
- 설명: 두 작품 모두 인간의 탐욕과 구조적 불평등을 비판적으로 다룸. 블랙코미디적 연출과 긴장감 있는 전개가 공통적이며, 사회풍자 중심 서사로 감정 톤 일치.

2. 버닝 → 자백
- 주요 공통 요소: 심리 스릴러 / 인물의 내면 혼란 / 미스터리 구조
- 적용 규칙: ①②④
- 설명: 감정의 불확실성과 인간 본성에 대한 탐구가 중심. 두 영화 모두 관객의 해석 여지를 남기는 열린 결말 구조와 음울한 색채 톤이 유사.

3. 비상선언 → 콘크리트 유토피아
- 주요 공통 요소: 재난 이후 인간 군상 / 폐쇄된 공간 내 갈등 / 집단 심리 묘사
- 적용 규칙: ②③⑥
- 설명: 대규모 재난 속 인간의 생존과 윤리를 탐구. 긴장감 있는 연출, 집단 이기주의·공포감 연출 방식이 일치하며, 사회적 메시지 중심 구조.

4. 암살 → 남한산성
- 주요 공통 요소: 역사적 배경 / 집단의 신념과 생존 / 리더십과 배신의 서사
- 적용 규칙: ①③⑤
- 설명: 모두 역사적 전란 속 인간의 도덕성과 결단을 다룸. 묵직한 톤과 웅장한 미장센, 공동체의 분열이라는 주제가 공통.

5. 광해, 왕이 된 남자 → 사도
- 주요 공통 요소: 왕권과 인간성의 갈등 / 부자·군신 관계 / 감정 중심 서사
- 적용 규칙: ②④⑥
- 설명: 권력과 감정 사이의 인간적 고뇌를 탐구. 사극임에도 인물 내면의 심리와 감정선을 세밀하게 그린다는 점에서 서사 구조 유사.

6. 동주 → 천문: 하늘에 묻는다
- 주요 공통 요소: 역사 속 실존 인물 / 철학적 대화 중심 / 시각적 정제미
- 적용 규칙: ②④⑤
- 설명: 두 작품 모두 조선 시대 실존 인물을 통해 인간의 내면과 신념을 탐구함. 시적 감수성과 잔잔한 미장센, 인물 간의 철학적 대화가 주를 이룸.

7. 천문: 하늘에 묻는다 → 사도
- 주요 공통 요소: 왕권과 인간성의 갈등 / 감정 서사 중심 / 역사적 배경의 비극성
- 적용 규칙: ①③⑤
- 설명: 왕과 신하, 부자 간의 관계를 감정선 중심으로 풀어내는 사극. 역사적 사실보다 인물의 내면 심리에 초점을 둔 연출이 유사.

8. 항거: 유관순 이야기 → 국제시장
- 주요 공통 요소: 시대적 고난 속 인내 / 공동체 정신 / 희생과 가족애
- 적용 규칙: ②⑤⑥
- 설명: 개인의 삶이 시대의 고난과 맞닿아 있다는 서사 구조가 같음. 사회적 메시지와 감정 몰입형 연출이 공통 요소.

9. 국제시장 → 남한산성
- 주요 공통 요소: 역사·시대의 선택 / 리더의 결단 / 공동체의 가치
- 적용 규칙: ③⑤⑥
- 설명: 시대적 배경 속에서 개인과 국가의 선택을 다룸. 인간적 고뇌와 책임, 역사적 리얼리즘 연출이 유사하게 작용.

10. 기생충 → 콘크리트 유토피아
- 주요 공통 요소: 계층 갈등 / 재난 이후 사회 구조 풍자 / 폐쇄 공간 서사
- 적용 규칙: ②④⑥
- 설명: 모두 사회 구조 속 불평등을 극단 상황에서 드러냄. 상징적 공간 연출(지하, 아파트 등)과 현실 풍자의 긴장감이 공통된 감정 톤을 형성.

11. 마녀 Part 1. The Subversion → 마녀 Part 2. The Other One
- 주요 공통 요소: 세계관·연출 동일(박훈정 감독) / 초능력 실험체 설정 / 액션 연출 톤 일치
- 적용 규칙: ①②④
- 설명: 감독 동일성과 세계관 연속성으로 캐릭터의 감정선, 폭발적 액션, 색감 톤이 그대로 이어짐. 감정+액션 융합형 스릴러 구조.

12. 추격자 → 악인전
- 주요 공통 요소: 연쇄살인 서사 / 범인-형사 대립구도 / 리얼리즘 기반 폭력 연출
- 적용 규칙: ②③⑤
- 설명: 현실적 공포를 극사실적 방식으로 표현. 잔혹 범죄를 통해 인간 본성을 드러내며, 스릴러적 리듬과 심리 긴장감 유사.

13. 더 폰 → 알몸의 시간 (가상 추천 예시)
- 주요 공통 요소: 시간 왜곡 장치 / 가족 구출 중심 / 몰입형 플롯 구조
- 적용 규칙: ③④⑥
- 설명: 과거와 현재의 교차를 통해 스릴을 구축하는 구조. 긴박한 시간 제약과 인물 심리의 붕괴가 공통 감정 톤으로 작용.

14. 모범시민 → 내부자들
- 주요 공통 요소: 정의·복수 서사 / 부패 권력 비판 / 사회 풍자적 스릴러
- 적용 규칙: ②⑤⑨
- 설명: 사회 구조를 비판하는 지적 스릴러. 폭력보단 심리적 대립과 언론·정치 풍자를 통해 서스펜스를 구축.

15. 독전 → 범죄도시
- 주요 공통 요소: 범죄조직 내부 갈등 / 카리스마형 주인공 / 리듬감 있는 액션 스릴
- 적용 규칙: ②③⑥
- 설명: 두 작품 모두 범죄 세계의 파워게임을 긴장감 있게 연출. 시각적 강도, OST 활용, 인물 중심 구도가 유사하며 감정 리듬 일치.

2. 시나리오 B – 사회 비판형 스릴러 선호 사용자

→ 추천: “콘크리트 유토피아”, “비밀은 없다”, “악인전”
(규칙②③⑥ 적용: 사회 구조 비판, 폐쇄 공간 갈등, 긴장 리듬 일치)
3. 시나리오 C – 범죄 액션 스릴러 선호 사용자
‘범죄도시’, ‘악인전’, ‘공조2’ 즐겨찾기
→ 추천: “독전”, “마녀 Part 1”, “범죄도시 2”
(규칙①②⑤ 적용: 액션 강도·감정 곡선 유사, 세계관 연속성)
4. 시나리오 D – 미스터리/서사형 스릴러 선호 사용자
‘세븐’, ‘컨페션’, ‘시간을 달리는 소녀(스릴러판)’ 좋아요
→ 추천: “컨페션”, “리멤버”, “더 폰”
(규칙②④⑨ 적용: 시간·기억 모티프, 반전 서사, 서정적 긴장감)
