# 추천 규칙 / 알고리즘 기획

## 1. 추천 알고리즘 설계 방향

- 긴장 곡선 (Tension Arc): 긴장·불안이 어떻게 축적→폭발→해소/미해소** 되는지 패턴을 추출하여 유사 곡선 매칭 
- 서사 구조 (Narrative Pattern): 추적형 / 수사형 / 음모 해소형 / 심리 압박형 / 생존·밀실형 / 법정 스릴러 등 구조 기반 유사도 평가
- 무드·톤 (Mood/Tone): 어둡·차갑·불길·음울** 등 시각 톤과 리듬(슬로우빌드/급전개), 조명·색채 대비, 카메라 시점의 결 유사도
반전 유형 (Twist Type): 결말 반전 / 인물 정체 반전 / 시점 반전 태그 일치도 반영 (복수 반전일수록 가중)
- 감독/연출 스타일: 컷 편집 속도, 대사 밀도, 사운드 텐션 활용, 현실주의 vs 스타일라이즈드 연출 경향 반영
- 공간/배경 세팅 폐쇄 공간(밀실·병원·집), 도시, 법정, 실시간 POV 등 배경 일치 시 가산        
- 사운드 디자인: 저음 드론, 심장박동형 리듬, 정적 활용 등 청각적 긴장 패턴 유사도 반영
- 결말 성향 (Ending Preference): 해소형(정의 실현) / 열린형(여운·불확실) / 비극형 중 사용자 선호 결말을 옵션으로 반영  
 - 자극 허용도 (Content Tolerance): 고어/점프스케어/잔혹 묘사 허용 여부를 필터·가중치로 반영 (비허용 시 감점/제외)


## 2. 기준
### 2-1. 콘텐츠 특성 기준
- 키워드 기반: ‘범죄’, ‘심리’, ‘추리’, ‘불안’, ‘반전’ 등 주요 키워드 유사도 분석
- 긴장도(Tempo) 구조: 이야기 전개의 리듬(서서히 고조되는 vs 급전개형)을 기반으로 유사한 긴장 흐름을 탐색
- 감독 연출 스타일: 컷 편집 속도, 대사 간격, 사운드 활용 방식 등 연출 템포 유사도 비교
- 공간/배경 톤: 폐쇄 공간, 도시, 법정 등 배경 특성과 색조 톤(어둡·차가움·냉정함 등) 일치 여부
- 서사 패턴(Story Type):  추적형 / 음모 해소형 / 심리 압박형 / 생존 밀실형 등 서사 구조 분류로 유사도 평가
### 2-2. 메타데이터 기준
- 제작사/플랫폼: 동일 제작 라인(예: Netflix Original, CJ ENM 등)일 경우 가산점 부여 
- 감독/배우: 동일 인물 또는 협업 라인 일치 시 신뢰도 및 톤 일관성 가산                 
- 국가/언어:  작품 특유의 연출 감각(예: 미국=범죄 서사 중심, 한국=심리 서스펜스 중심)을 고려    
- 상영 형태: OTT 오리지널, 극장 개봉작, 미니시리즈 등 형식별 텐션 차이 반영             
- 결말 성향:  해소형(정의 실현) / 열린형(불완전한 진실) / 비극형(허무감 중심)으로 구분       
### 2-3. 사용자 선호 기준
- 좋아요(Like): 사용자가 선호한 스릴러의 서사 패턴·감독·톤 특성 가중치  +1      
- 싫어요(Dislike): 유사한 반전 구조나 불쾌감 높은 톤의 작품은 감점 −1.5     
- 시청 완료율(Completion): 완주율 높은 콘텐츠의 서사·긴장 패턴 중심으로 추천 확률 강화         
- 중도 이탈(Abandon): 이탈률 높은 구조(예: 과도한 고어, 급전개 서사)는 감점 −1   
- 재시청/즐겨찾기: 반복 시청 콘텐츠의 연출 톤·감정 템포를 반영해 가산 +1       
- 검색·클릭 로그: 탐색 행동(검색, 미리보기, 예고편 시청 등)에 따라 관심 주제 가중치 보정
