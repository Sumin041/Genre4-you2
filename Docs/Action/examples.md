## 5. 추천 예시 (10개)
 1. 존 윅 3 → 존 윅 4
- 주요 공통 요소: 시리즈 연속성, 주인공 성장, 타격감 강한 전투 
- 적용 규칙: ①, ②, ⑧
- 설명: 시리즈 연속성 유지, 전투 스타일 및 주인공 성장 서사 일치

 2. 익스트랙션 → 익스트랙션 2
- 주요 공통 요소: 구출 미션, 액션 리듬, 고속 전개
- 적용 규칙: ②, ③, ⑧
- 설명: 액션 템포와 전투 스타일이 거의 동일하며, 시리즈 연속성 반영

 3. 매드맥스: 분노의 도로 → 분노의 도로 스핀오프
- 주요 공통 요소: 포스트아포칼립스 세계관, 차량 액션, 폭발적 연출 
- 적용 규칙: ②, ③, ⑧
- 설명: 세계관과 차량 중심 액션, 강렬한 비주얼 톤 유사

4. 더 그레이 맨 → 존 윅 4
- 주요 공통 요소: 첩보 액션, 총기전, 고독한 주인공
- 적용 규칙: ②, ③, ⑤
- 설명: 타격감 강한 총격 액션, 시각·OST 톤 유사

5. 로스트 불렛 → 로스트 불렛 2

- 적용 규칙: ②, ③, ⑨
