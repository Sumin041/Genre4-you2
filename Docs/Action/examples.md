## 2. 사용자 선호 시나리오

‘길복순’, ‘더 라운드업’, ‘독전 2’에 좋아요

설명: 주인공 성장 서사와 팀플레이·전투 리듬 일치, 감독 스타일 및 감정선 유사
2. 시나리오 B – 다크 액션/첩보 선호 사용자

→ 추천: “존 윅 4”, “익스트랙션 2”, “폴라”

설명: 격투·총격 액션 강도와 주인공 감정 곡선 일치, 전투 템포 및 긴장감 반영
3. 시나리오 C – 팀플레이/유머 액션 선호 사용자
