
# 액션 장르 개요

## 1. 장르 정의
 - 액션(Action)은 신체적 움직임, 갈등, 전투, 추격 등 속도감과 긴장감을 핵심으로 하는 장르이다.
 - 시청자는 스토리보다 시각적 몰입감과 박진감 있는 전개를 중시한다.

## 2. 주요 특징
 - 고속 전투 : 전투, 추격, 폭발 등 빠른 장면 전환
 - 주인공 중심 서사 : 영웅, 복수, 생존 등의 단일 목표 구조
 - 시각 효과 중심 : 카메라 워크, 편집, 특수효과 중요
 - 감정보다는 긴장감: 서사보다 에너지와 속도감에 초점

## 3. 시청자 특성
 - 몰입형 시청자 : 긴장감과 전투 장면 중시
 - 영웅서사형 : 정의, 복수, 리더십 중심의 이야기 선호
 - 현실감형 : 과도한 초능력보다는 리얼한 액션 선호

## 요약
 - 액션 장르는 시각적 긴장감과 속도감을 핵심으로 한다.
 - 추천은 전투 스타일, 연출 템포, 시리즈 연결성을 기반으로 설계하며,
 - 시청자의 몰입 패턴 분석이 중요하다.
