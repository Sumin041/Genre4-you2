## 1. 사용자 정의 
**OTT 플랫폼 사용자**
  - 넷플릭스, 티빙의 다양한 OTT 서비스를 사용하는 일반 소비자
  - 플랫폼 추천 기능을 적극적으로 활용하는 10대~40대 중심
  - 감정 곡선, 장르, 감독/배우 성향에 따라 콘텐츠를 고르는 경향

## 2. 사용자로부터 받아야 할 정보 
- **시청 기록**
  - 최근 시청한 작품
- **선호 장르**
  - 직접 선택 또는 시청 이력 기반 추출
- **선호 감정 분위기**
  - 예: 잔잔한, 감동적인, 긴장감 있는 등
- **시청 후 추천 의향 정보**
  - 사용자가 작품을 끝까지 시청한 후,
    “이 작품을 다른 사용자에게 추천할 의향이 있는지” 선택할 수 있는 옵션 제공 
        - 선택 옵션 예:
            - 좋아요
            - 싫어요

## 3. 사용자에게 전달하고자 하는 정보
- **추천 작품 3개**
  - 사용자 선호 기반 장르 내에서 유사한 감정곡선/분위기 작품
  - 같은 감독/제작사/주연배우의 작품
  - OTT 내 현재 시청 가능한 콘텐츠로 한정


## 4. 시청 환경(상황)에 따른 처리 방식

### 4.1 영상 전체 시청(완시청) 환경
1. **완시청 기준**
  - 작품 러닝타임의 일정 비율(예: 80% 이상)을 시청한 경우를 ‘완시청’으로 간주
2. **처리 방식**
  - 완시청 시, 시청 종료 화면에서 “이 작품을 다른 사용자에게 추천하시겠습니까?” 문구와 함께
    좋아요 / 싫어요 선택 옵션을 노출
  - **좋아요 선택 시**
    - 해당 작품에 높은 가중치의 긍정 신호 부여  
      - 예: 작품 추천 점수 +1 
      - 동일 장르/감정 분위기/감독/배우의 작품에도 소량의 가산점 부여(간접 긍정 신호)
  - **싫어요 선택 시**
    - 해당 작품에 높은 가중치의 부정 신호 부여  
      - 예: 작품 추천 점수 −1.5  
      - 유사한 감정 분위기나 설정을 가진 작품의 점수도 일부 감소시켜, 향후 추천에서 노출 빈도 감소

### 4.2 영상 중간까지 시청(부분 시청) 환경
1. **부분 시청 기준**
  - 러닝타임의 일정 비율(예: 30% 이상 80% 미만)만 시청한 경우를 ‘부분 시청’으로 간주
2. **처리 방식**
  - 부분 시청 상태에서는 다음 두 가지 경우를 구분해 처리한다.

  1) **부분 시청 + 좋아요/싫어요 평가가 있는 경우**
     - 사용자가 중간에 시청을 멈추더라도,  
       “좋아요” 또는 “싫어요”를 명시적으로 선택했다면,
       - 해당 선택을 완시청과 동일한 수준의 선호/비선호 정보로 반영  
       - 예:  
         - 좋아요 → 작품 점수 +1  
         - 싫어요 → 작품 점수 −1.5 

  2) **부분 시청 + 평가(좋아요/싫어요) 없이 이탈한 경우**
     - 이 경우에는 사용자의 의도가 불분명하므로,  
       강한 선호나 비선호로 해석하지 않고 ‘중립적인 신호’로 처리한다.
    - 중립적인 신호: 시청 기록은 저장하되,추천 점수에 아주 낮은 가중치 또는 0에 가까운 가중치로만 반영하는 것을 의미한다.
     - 예시 규칙:
       - 시청 시간 비율에 따라 작은 수준의 참고용 점수만 반영  
         - 30~50% 시청: 작품 점수 +0.3 (관심이 있을 수 있는 약한 후보로만 취급)  
         - 50~80% 시청: 작품 점수 +0.5 (관심도가 어느 정도 있다고 보되, 좋아요만큼 강하게 반영하지 않음)
       - 좋아요/싫어요가 없는 이상,  
         - 해당 작품을 강하게 밀어주지도(추천 상단 배치 X) 강하게 제외하지도 않는 ‘약한 후보’ 정도의 위치로 유지
     - 동일 작품을 나중에 다시 재생하여 완시청한 경우,
       - 최종적으로 완시청 규칙(4.1)을 우선 적용하고,  
       - 부분 시청 당시의 중립 신호보다 완시청 + 평가 정보가 더 높은 우선순위를 갖는다.

### 4.3 잘못 클릭하여 재생한 환경(오클릭)
1. **오클릭 기준**
  - 러닝타임의 매우 짧은 구간(예: 전체의 30% 미만 또는 3분 미만)만 시청 후 바로 종료한 경우를
    ‘잘못 클릭(오클릭)’으로 간주
2. **처리 방식**
  - 오클릭으로 판단된 시청 이력은 선호 정보로 사용하지 않고, 추천 점수 계산에서 완전히 제외한다.
  - 단, 내부 로그에는 “재생 시도” 수준의 기록만 남길 수 있으나, 추천 알고리즘의 학습에는 사용하지 않는다. 
  - 이후 동일 작품을 다시 재생하여 일정 비율 이상 시청할 경우, 신규 시청 이력을 기준으로 부분 시청 규칙(4.2) 또는 완시청 규칙(4.1)을 적용하고,기존 오클릭 이력은 무시한다.
