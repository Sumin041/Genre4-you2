# Genre4-you2 Framework (공통 문서 양식)

이 문서는 팀의 모든 장르 문서(`thriller.md`, `animation.md`, `romance.md`, `action.md`)가  
공통으로 따라야 하는 **작성 틀(template)**을 정의합니다.

—

## 문서 구성 구조
각 장르 문서는 아래의 동일한 구조를 따라 작성합니다:


# [장르명] Genre4-you2 기획

## 1. 목표
- 이 장르에서 추천 알고리즘이 해결하고자 하는 문제는 무엇인가?
- “특정 콘텐츠를 본 사용자가 다음에 무엇을 볼까?”를 어떤 관점으로 풀 것인가?

## 2. 기준 
- 콘텐츠 특성 기준 (예: 키워드, 분위기, 감독, 배우 등)
- 메타데이터 기준 (예: 제작사, 시즌, 연령등급)
- 사용자 선호 기준 (예: 좋아요, 싫어요)

## 3. 로직 예시 (간단한 규칙)
  - 규칙 ①: 같은 감독/제작사 → 가산점  
  - 규칙 ②: 비슷한 키워드(“추리”, “감정”) → 가산점  
  - 규칙 ③: 사용자 '좋아요' 선택 -> 해당 장르 가산점
  - 규칙 ④: 사용자 '싫어요' 선택 -> 해당 장르 감점
