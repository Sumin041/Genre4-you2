# 추천 결과 예시 (Top~3)

## 1. 추천 예시 (10개)
1. 나의 해방일지 -> 멜로가 체질
- 주요 공통 요소: 일상 감정선 / 현실적 대사 / 감정의 여백
- 적용 규칙: ②,③,⑨
- 설명: 감정을 크게 드러내지 않고 ‘여백’으로 표현하는 연출 톤이 동일함

2. 사랑의 불시착 -> 별에서 온 그대
- 주요 공통 요소: 운명적 로맨스 / 판타지 설정
- 적용 규칙: ②,③,④
- 설명: 비인간적 존재와 인간 관계에서 감정 고조가 서사의 핵심으로 작동

3. 스물다섯 스물하나 -> 청춘기록
- 주요 공통 요소: 청춘 성장 / 현실과 이상 사이의 감정선
- 적용 규칙: ②,③,⑨
- 설명: 꿈·진로·사랑이 서로 충돌하며 성장하는 감정 곡선이 유사

4. 그 해 우리는 -> 연애의 발견
- 주요 공통 요소: 이별 → 회고 → 재회 구조
- 적용 규칙: ②,⑨
- 설명: 시간이 흐른 뒤 감정을 다시 바라보는 서사 패턴 동일

5. 갯마을 차차차 -> 사내맞선
- 주요 공통 요소: 명확한 호감 표현 / 밝고 경쾌한 로코 톤
- 적용 규칙: ③,⑤
- 설명: 빠르고 산뜻한 감정 전개가 공통적으로 나타남

6. 유미의 세포들 -> 이번 생도 잘 부탁해
- 주요 공통 요소: 감정 내면 묘사 / 자기 이해 과정 중심
- 적용 규칙: ③,⑨
- 설명: 감정을 ‘해석’하고 받아들이는 과정이 서사의 동력으로 작동

7. 도깨비 -> 푸른 바다의 전설
- 주요 공통 요소: 초월적 존재 × 인간 / 감정 성장 서사
- 적용 규칙: ①,②,③,④
- 설명: ’운명적 사랑’ 감정선과 OST 사용 방식까지 유사

8. 이번 생은 처음이라 -> 도시남녀의 사랑법
- 주요 공통 요소: 현실 연애 관점 / 관계 철학 중심
- 적용 규칙: ②,⑨
- 설명: 사랑보다 ‘사랑을 바라보는

9. 하트시그널 4 -> 환승연애 3
- 주요 공통 요소: 실제 감정 변화 / 선택과 관계 재구성
- 적용 규칙: ⑤,⑥
- 설명: 감정의 미세 흔들림 자체가 시청 포인트인 관찰형 로맨스

10. 이 연애는 불가항력 -> 도깨비
- 주요 공통 요소: 운명적 인연 / 감정 고조 곡선 동일
- 적용 규칙: ②,④
- 설명: 외부적 ‘운명 요인’이 감정선의 방향성을 결정하는 서사 구조

11. 태양의 후예 -> 사랑의 불시착
- 주요 공통 요소: 군/국가/위험 상황 속 로맨스 / 강한 커플 케미
- 적용 규칙: ②,③,⑤
- 설명: 외부 환경(위험/임무)이 두 사람의 감정 고조를 강화하는 구조

12. 이상한 변호사 우영우 -> 뷰티 인사이드
- 주요 공통 요소: 서로 다름을 이해하는 관계 / 존중 기반 로맨스
- 적용 규칙: ②,③,⑥
- 설명: 사랑의 핵심을 ‘이해의 과정’으로 다루는 서사

13. 로맨스는 별책부록 -> 김비서가 왜 그럴까
- 주요 공통 요소: 직장 로맨스 / 일+사랑 병행
- 적용 규칙: ③,④,⑤
- 설명: 일상 공간 속에서 관계가 자연스럽게 발전하는 구조

14. 동백꽃 필 무렵 -> 이번 생은 처음이라
- 주요 공통 요소: 서툴지만 진심인 관계 / 감정 정돈 과정
- 적용 규칙: ②,③,⑨
- 설명: 큰 사건보다 ‘서로를 이해해가는 감정선’이 중심

15. 호텔 델루나 -> 도깨비
- 주요 공통 요소: 판타지 설정 / 한정된 시간 속 사랑
- 적용 규칙: ②,④
- 설명: 비현실적인 세계관이 감정의 깊이를 강조하는 서사 구조


## 2. 사용자 선호 시나리오 
1. 시나리오 A — 잔잔·치유 멜로 선호 사용자
‘나의 해방일지’, ‘멜로가 체질’, ‘사랑의 이해’를 좋아요
→ 추천: “동백꽃 필 무렵”, “이번 생은 처음이라”, “가족입니다”
(규칙 ②③⑨ 적용: 감정의 여백 / 현실적 대사 / 서사 내면화 기반)

2. 시나리오 B — 설렘·로코 중심 사용자
‘갯마을 차차차’, ‘사내맞선’, ‘김비서가 왜 그럴까’를 즐겨찾기
→ 추천: “그녀의 사생활”, “환승연애”, “하트시그널”
(규칙 ③⑤ 적용: 명확한 호감 표현 + 가벼운 유쾌 톤 기반 추천)

3. 시나리오 C — 슬로우번(서서히 가까워지는 감정) 선호 사용자
‘그 해 우리는’, ‘도시남녀의 사랑법’, ‘연애의 발견’ 완시청
→ 추천: “남남”, “사랑이라 말해요”, “지금, 헤어지는 중입니다”
(규칙 ②⑨ 적용: 감정 회고형 서사 + 시간 흐름에 따른 감정 성숙)

4. 시나리오 D — 판타지/운명 서사 선호 사용자
‘도깨비’, ‘환혼’, ‘마이 데몬’을 좋아요
→ 추천: “푸른 바다의 전설”, “호텔 델루나”, “이 연애는 불가항력”
(규칙 ②④ 적용: 초월 세계관 + ‘운명’이 감정선에 개입하는 구조)
