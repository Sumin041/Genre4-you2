# 추천 규칙 / 알고리즘 기획

## 1. 추천 알고리즘 설계 방향
- 반전 구조 매칭: 유사한 플롯 트위스트 기반 추천
- 감정 템포 분석: 긴장도 변화 곡선(불안→안정→충격) 유사도 기반
- 배경 유형 분류: 도시/법정/고립 공간 등 배경 일치 시 가중치 증가
- 연출 톤 분석: 슬로우빌드형 vs 급전개형 구분

## 2.기준
### 2-1. 콘텐츠 특성 기준 
- 키워드 기반: 감정(‘성장’, ‘희생’, ‘가족’) 및 장르(‘판타지’, ‘모험’) 키워드 유사도 분석
- 분위기(톤): 색채, 작화 스타일,감정선 유사도 분석 (예:밝음·감성적·어두움 등)
- 감독 스타일	특정 감독의 연출 리듬, 대사 밀도, 음악 사용 패턴 기반
- OST/사운드: 서정적·웅장한 등 음악적 분위기가 비슷한 작품 우선 추천
### 2-2. 메타데이터 기준
- 제작사: 동일 스튜디오(예: 픽사, 지브리, 유포테이블)일 경우 가산점 부여
- 국가/언어: 일본·미국·한국 등 국가별 제작 특성 고려
- 연령등급:	사용자의 시청 연령대와 일치 시 우선 추천
- 상영 형태: OTT 오리지널 vs 극장판 여부로 분류
- 시즌·세계관 연속성: 시리즈 후속작 또는 같은 세계관 공유 시 연결 추천
### 2-3. 사용자 선호 기준
- 좋아요: 사용자가 선호한 작품의 키워드 및 감독 특성 가중치 +1
- 싫어요: 해당 작품과 유사한 키워드·감정선 감점
- 시청 완료율: 끝까지 본 작품의 특징을 중심으로 추천 확률 강화
- 재시청/즐겨찾기: 재시청한 작품과 유사한 톤·연출의 콘텐츠 우선순위 상승
- 검색·클릭 로그: 관심 표현(예: 검색, 미리보기 재생 등)에 따라 가중치 미세 조정
