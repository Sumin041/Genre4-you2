# 추천 결과 예시 (Top~3)

## 1.추천 예시 (15개)
1. 너의 이름은 -> 날씨의 아이
- 주요 공통 요소: 감독 동일(신카이 마코토), 감성 톤 유사	
- 적용 규칙 : ①,②,④
- 설명: 신카이 마코토 감독, 감정선·OST 일치
2.  센과 치히로의 행방불명 -> 모노노케 히메
- 주요 공통 요소: 제작사 동일(지브리) / 환경 주제
- 적응 규칙: ①,②
- 설명: 자연·신화적 세계관 공유
3. 코코 → 소울
- 주요 공통 요소: 픽사 제작, 삶과 예술을 다룬 감동 서사
- 적용 규칙: ①,②,③
- 설명: 감정 곡선(죽음→깨달음→감동)이 유사하며 음악 테마가 공통됨
4. 귀멸의 칼날: 무한열차편 → 주술회전 0
- 주요 공통 요소: 판타지 액션, 성장 서사
- 적용 규칙: ②,③,⑨
- 설명: 전투 연출의 강도와 감정 밀도가 유사한 다크 판타지 계열 작품
5. 인사이드 아웃 → 주토피아
- 주요 공통 요소: 감정 중심, 사회적 메시지
- 적용 규칙: ②,⑤,⑨
- 설명: 감정 시각화 구조와 사회적 가치 서사가 공통적으로 나타남
6. 보스 베이비 → 슈퍼배드
- 주요 공통 요소: 가족 코믹물, 밝은 색채
- 적용 규칙: ②,④,⑤
- 설명: 가족 중심 서사와 OST 분위기, 코믹 감정선이 유사함
7. 루카 → 엘리멘탈
- 주요 공통 요소: 성장, 우정, 픽사 감성
- 적용 규칙: ①,②,③,⑤
- 설명: 감정 곡선과 제작사 특유의 따뜻한 색감이 동일함
8. 드래곤 길들이기 → 쿵푸팬더
- 주요 공통 요소: 성장, 훈련 서사, 드림웍스 제작
- 적용 규칙: ①,②,③
- 설명: 캐릭터의 성장과 자기 극복 서사가 유사함
9. 토이 스토리 3 → 몬스터 주식회사
- 주요 공통 요소: 우정, 이별, 픽사 감성
- 적용 규칙: ①,②,③,⑤
- 설명: 감정 곡선 및 OST 분위기가 유사한 픽사 대표 감동작
10. 명탐정 코난: 흑철의 어영 → 루팡 3세 vs 명탐정 코난
- 주요 공통 요소: 시리즈·세계관 연속성
- 적용 규칙: ①,⑧
- 설명: 동일 캐릭터와 세계관을 공유하는 확장판 스핀오프 구조
11. 버즈 라이트이어 → 월-E
- 주요 공통 요소: SF, 철학적 주제
- 적용 규칙: ①,②,⑨
- 설명: 인간과 기계, 존재 의미를 다루는 픽사식 미래 서사 구조
12. 모아나 → 라푼젤
- 주요 공통 요소: 디즈니 제작, 자아 찾기 서사
- 적용 규칙: ①,②,③,⑤
- 설명: 여성 주인공의 성장과 음악적 감정선이 유사한 디즈니 대표작
13. 슈렉 2 → 메가마인드
- 주요 공통 요소: 패러디, 유머, 드림웍스 제작
- 적용 규칙: ①,②,④
- 설명: 패러디 중심 코믹 요소와 음악 톤, 연출 감각이 유사함
14. 마당을 나온 암탉 → 태일이
- 주요 공통 요소: 한국 애니메이션, 사회적 현실 메시지
- 적용 규칙: ①,②,⑩
- 설명: 현실 사회 문제를 성장 서사와 감정선 중심으로 표현함
15. 스파이더맨: 스파이더버스 → 틴 타이탄스 고! 더 무비
- 주요 공통 요소: 슈퍼히어로, 코믹톤
- 적용 규칙: ②,③,⑦
- 설명: 빠른 컷 전환과 색감, 유머 코드가 비슷한 히어로 애니메이션

## 2.사용자 선호 시나리오
1. 시나리오 A – 감성 성장물 선호 사용자
‘코코’, ‘루카’, ‘소울’에 좋아요
→ 추천: “엘리멘탈”, “너의 이름은”, “날씨의 아이”
(규칙①②⑤ 적용: 감정·OST 유사, 감독 동일성 가중치)
2. 시나리오 B – 다크 판타지/액션 선호 사용자
‘귀멸의 칼날’, ‘체인소맨’, ‘주술회전 0’ 완시청
→ 추천: “진격의 거인”, “블루 록”, “나루토 극장판”
(규칙②③⑨ 적용: 전투 강도·감정 곡선 일치)
3. 시나리오 C – 가족/코믹 중심 사용자
‘슈퍼배드’, ‘보스 베이비’, ‘토이 스토리’ 즐겨찾기
→ 추천: “몬스터 주식회사”, “슈렉 2”, “주토피아”
(규칙①②④⑤ 적용: 제작사 동일 + 밝은 감정선 기반)
4. 시나리오 D – 철학적 주제 선호 사용자
‘월-E’, ‘소울’, ‘바다의 노래’ 좋아요
→ 추천: “태일이”, “버즈 라이트이어”, “모노노케 히메”
(규칙②③⑨ 적용: 존재 의미·감정 깊이 기반 추천)
